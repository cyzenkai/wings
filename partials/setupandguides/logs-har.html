Getting HAR Logs

  1. Open a Chrome browser, and press the option button at the top right corner.
  2. Select More Tools then click Developer Tools.
  3. Select Network tab, make sure recording is enabled (Red button).
  4. Log in as User then go to the CVR timeline. Try to play back the timeline.
  5. From the Developer Tools, right click the playlist item. If playlist does not exist, select the last item then choose the option Save as HAR with content.
You have successfully saved the HAR file logs.
