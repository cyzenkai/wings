No Recording in Library
  1. Check and make sure customer has rule scheduled and camera is armed for recording
  2. What is the internet speed (Wi-Fi/ LTE)?
  3. What Plan does the customer use?
    Note: if customer is on Basic plan, he will not get any recording for Ultra, Pro3 and Video Doorbell as it requires to purchase the paid plan
  4. Check and see if customer uses local storage and he sees recording in there
  5. Does the issue occur on admin or friend account?
  6. Check and see if the manual recording immediately uploaded to the library?
  7. Make sure customer doesn’t have any filter set in library
  8. Make sure and check no known issues or check, https://status.arlo.com/
  9. Make sure and check other clients (Web/Android/iOS)
  10. Screenshots of library
Recording disappeared/missing from Library
  1. Check and make sure it is working as expected as per customer’s subscription plan
  2. What plan does the customer use?
  3. Make sure and check no known issues or check, https://status.arlo.com/
  4. Make sure customer doesn’t have any filter set in library
  5. Make sure and confirm if customer previously saw the recording in the library
  6. Make sure if other people don’t have access to the same Arlo account
  7. Make sure friends account with admin rights has not deleted the video from Library
  8. Make sure and check other clients (Web/Android/iOS)
  9. Screenshots of library
  10. Collect the timestamp of the missed/delayed recording

Push Notification Issues
  1. Check if PN are enable in camera modes and armed
  2. Check phone settings to see if PN is enabled for Arlo app
  3. Check phone power setting mode and see if its disable
  4. Check if phone has internet connection (Cellular or Wi-Fi)
  5. Log out of the Arlo app and log back in (Do not just minimize the app, but explicitly log out and back in)
  6. Attempt to trigger motion and monitor for a push notification
    (Note: if user receive a push notification at this point, you do not need to proceed to the next step)
  7. Toggle push notification off and then back on and repeat the steps 1-4
  8. If still doesn’t work, then escalate the case.

Partner Integration Issues

  2. Confirm if the issue can be reproduced on the Arlo App (Live Stream)
  3. Rule out authentication issue for Alexa and Google home integration (link and unlink account)
    1. For Google home check out this KB, https://www.androidcentral.com/how-link-and-unlink-smart-device-services-google-homeapp
    2. For Alexa check out this KB, https://www.amazon.com/gp/help/customer/display.html?nodeId=201749270
    3. For Homekit, check out this KB, https://kb.arlo.com/000062279/How-do-I-connect-my-Arlo-camera-to-Apple-HomeKit
  4. Latency - request times out:
    1. Alexa, google and Homekit time out possible cause
    2. Rule out ISP connectivity issues at the time
    3. Add a screenshot of speedtest.net
  5. Check configuration
    1. For Alexa, Ensure Alexa is configured correctly, and the camera name can be recognized: https://kb.arlo.com/000044726/How-can-I-use-Amazon-devices-with-my-Arlo-cameras
    2. For Google Home, Add the Arlo device to a Room when live streaming from Google assistant: https://kb.arlo.com/000055108/How-do-I-connect-my-Arlo-cameras-to-the-GoogleAssistant-app
    3. For Homekit, follow below KB, https://kb.arlo.com/000062279/How-do-I-connect-my-Arlo-camera-to-Apple-HomeKit
  6. If issue with Arlo Video/Audio Doorbell announcements ensure to check the configuration is done correctly
    1. Issue with Alexa, follow below KB, https://kb.arlo.com/000062192/How-can-I-use-my-Arlo-Doorbell-with-Amazon-Alexa
  7. Check configuration
    1. If issue is related to Motion Announcements please check configuration is done correctly: https://kb.arlo.com/000062277/How-do-I-trigger-an-Amazon-Alexa-device-when-motion-isdetected-on-my-Arlo-camera
    8. If issue still persist collect below information,
    1. Check and make sure Arlo app is connected to 3rd party app by login into 3rd party app.
    2. To isolate the issue, make sure issue doesn’t exist with Arlo app (iOS/Android/Web)
    3. Check with L3 to see if there is any known outage
    4. Check to see if customer already contacted 3rd party support and if issue persist, collect steps to reproduce the issue
    5. Check 3rd party app is updated and collect the current app version customer is on

Battery doesn’t charge
  1. Make sure customer uses correct power adapter and cable that came with the box (Pro/Pro2: Quick charger 9V, Ultra: Regular charger 5V)
  2. For Battery doesn’t charge, ask customer to use different power outlet
  3. Ask customer to try and swap batteries with working camera
  4. Check the camera inside if it has physical damage (check flex cable inside camera)
  5. Check camera connectors (Charging port) for debris and corrosion
  6. Check the USB end connected to power adapter plugged in correctly
  7. Check battery polarity and remove – reinsert battery
  8. What type and what brand battery customer using (Applies to Gen3 camera only)
  9. If issue still occurs, create RMA based on the step 3. (If issue with battery create Battery RMA, if issue with camera create Camera RMA)
  10. Include all the above troubleshooting in case notes and escalate
Battery Drain
  1. Swap camera and battery and see if issue fixes to rule out issue with battery or camera
  2. Try changing camera location with working camera which doesn’t have issue
  3. Check battery dashboard/ StatMiner
    1. Check manual streaming
    2. Check triggered streaming
    3. Check RSSI
    4. Check total amount of streaming
    5. Check how noisy environment (Wi-Fi Scan results)
  4. Check camera settings
    1. Check sensitivity level
    2. Check length of recording
    3. Check power management settings is set to best battery life
    4. Check spotlight, color, behavior, brightness, night vision is affecting battery life
  5. Check how far camera from BS/SH
  6. Check any obstacles between camera and BS/SH
  7. Check home environment and network topology and other smart devices are on 2.4 affecting
  8. List of wireless devices customers has (Any new devices added to home, like printer, etc.)
  9. When did issue started, any specific time frame, after new firmware update, etc.
  10. Check how often customer live streaming
Battery DoA (Dead on Arrival)
  1. Make sure customer uses correct power adapter and cable that came with the box to charge battery (Pro/Pro2: Quick charger 9V, Ultra: Regular charger 5V)
  2. Ask customer to use different power outlet
  3. Swap camera and battery and see if issue fixes to rule out issue with battery or camera
  4. Issue RMA for battery/Camera and escalate the case to L3 with details
